2. Computed Value

nó sẽ tính toán từ 1.2em auto... thành Absolute value (tính toán giá trị tương đối sang giá trị tuyệt đối)

3. Used Value

vd: 2 thẻ div... ta quy định 1 thẻ trong là 50% thì nó sẽ thay đổi kích thước phụ thuộc vào thẻ cha của nó.(giai đoạn này dịch html thành dạng cây dom)
