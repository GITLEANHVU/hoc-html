Thẻ Unorder List không sắp xếp theo thứ tự Order List có sắp xếp theo thứ tự Description List giống như mô tả cho các thuật ngữ.

Danh sách Unorder List

Thẻ ul li không được đặt trong thẻ p.

  • list item 1
  • list item 2
  • list item 3
  • list item 4

Danh sách Order List

Thẻ ol li không được đặt trong thẻ p.

  1. list item 1
  2. list item 2
  3. list item 3
  4. list item 4

Danh sách Description List

Dùng để mô tả thông tin ai đó, hoặc thông tin chi tiết một thuật ngữ nào đó ...

Description title 1
Value 1
Description title 2
Value 1
Value 2
Value 3
Value 4
